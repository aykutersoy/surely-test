# TODO App in Symfony 3 #

A basic todo app in symfony 3

### How do I get set up? ###

* Configuration
    ```
    make deploy
    ```
* Dependencies
    - docker - https://docs.docker.com/install/
    - composer - https://getcomposer.org/doc/00-intro.md#installation-linux-unix-osx
* Deployment instructions
    - To run web server
    ```
    make run
    ```
